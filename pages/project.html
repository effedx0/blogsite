Discord bot

Throughout the process of developing a Discord bot, I followed a structured approach to enhance its functionality step by step. First, I set up the necessary environment by installing Node.js and the discord.js library, laying the foundation for the bot. Then, I implemented basic commands that allowed the bot to interact with users, starting with a simple ping command to test the connection and adding welcome and farewell messages to interact with new and leaving members on the server. To make the bot more dynamic, I added commands to tell random jokes and display user and server information. Additionally, I created a profanity filter to monitor and manage messages and implemented the ability to send bulk messages to users with specific roles.
